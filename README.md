# moonchild-tools
A few helpful tools for compressing/decompressing Moonchild art. You have the power to be his friend!

## fastfile_decompress.py
Decompresses fastfile archives, like "mc.art". Takes the following parameters:
```
Usage ./fastfile_decompress.py [mc.art] [outputdir]
```

## fastfile_compress.py
Recompresses fastfile archives, like "mc.art", and produces a byte-accurate replication. Takes the following parameters:
```
Usage ./fastfile_decompress.py [inputdir] [outfile]
```
